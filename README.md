<p align="center">
A web platform that allows readers to create communities, contribute financially to purchase books, and share them through a reservation system.
</p>

<hr>

<h2>🚀 Project Overview</h2>

<p>
The <b>Collaborative Community Library</b> is a platform where readers can build a shared library by contributing financially and participating in book selection.
</p>

<ul>
<li>Create or join reading communities</li>
<li>Contribute monthly to buy books</li>
<li>Suggest and vote for books</li>
<li>Reserve books</li>
<li>Earn participation points</li>
</ul>

<hr>

<h2>🎯 Project Objectives</h2>

<ul>
<li>Allow users to <b>create and join communities</b></li>
<li>Enable <b>financial contributions</b></li>
<li>Allow members to <b>suggest and vote for books</b></li>
<li>Implement a <b>book reservation system</b></li>
<li>Create a <b>points-based participation system</b></li>
<li>Provide <b>transparent budget management</b></li>
</ul>

<hr>

<h2>👥 System Users</h2>

<h3>1️⃣ Visitor</h3>

<p>A non-registered user who can explore the platform.</p>

<ul>
<li>Browse the platform</li>
<li>View public communities</li>
<li>Create an account</li>
</ul>

<h3>2️⃣ Member</h3>

<p>A registered user who joined a community.</p>

<ul>
<li>Join or leave communities</li>
<li>Pay monthly contributions</li>
<li>Suggest books</li>
<li>Vote for books</li>
<li>Reserve books</li>
<li>Write book reviews</li>
<li>View personal points</li>
</ul>

<h3>3️⃣ Community Administrator</h3>

<ul>
<li>Create communities</li>
<li>Manage members</li>
<li>Validate book purchases</li>
<li>Manage the budget</li>
<li>Manage reservations</li>
</ul>

<h2>📚 Reservation System</h2>

<ul>
<li>View available books</li>
<li>Reserve a book</li>
<li>Join waiting list</li>
<li>Return borrowed books</li>
</ul>

<p>
The system automatically manages:
</p>

<ul>
<li>Book availability</li>
<li>Borrow duration</li>
<li>Waiting queues</li>
<li>Reservation priority</li>
</ul>

<hr>

<h2>⭐ Points System</h2>

<p><b>Members earn points when:</b></p>

<ul>
<li>Paying contributions</li>
<li>Returning books on time</li>
</ul>

<p><b>Members lose points when:</b></p>

<ul>
<li>Returning books late</li>
</ul>

<p>
Points give advantages like:
</p>

<ul>
<li>Reservation priority</li>
<li>Extended borrowing time</li>
</ul>

<hr>

<h2>🚫 Reservation Restrictions</h2>

<ul>
<li>Members with <b>negative points</b> cannot reserve books</li>
<li>Members who <b>haven’t returned borrowed books</b> cannot reserve</li>
</ul>
